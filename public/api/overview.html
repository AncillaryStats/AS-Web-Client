Overview

www.ancillaryapi.comis the base url for the AncillaryStats API

All method calls should be GET requests to a specific endpoint. The API returns JSON in one of two formats:

Successful response:

{
  "status": <status code>,
  "result": <response object>
}

Error response:

{
  "status": <status code>,
  "message": "<error message>",
}
