NFL Teams

URL

GET http://www.ancillaryapi.com/api/<sport>/teams

Sport

nfl

Response

JSON object. Top level API result - is used to discover team IDs, and through those the team rosters containing player IDs

Example Result

{
  "status": 200,
  "result": [
    {
        "division": "1-0, 1st in AFC North",
        "id": 1,
        "name": "Cincinnati Bengals"
    },
    {
        "division": "0-1, 2nd in AFC North",
        "id": 2,
        "name": "Baltimore Ravens"
    },
    {
        "division": "0-1, 2nd in NFC North",
        "id": 3,
        "name": "Minnesota Vikings"
    },

    ...
  ]
}

nfl

nfl
